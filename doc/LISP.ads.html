Description

Package LISP is
PACKAGE LISP This package serves as the base package for the interpreter, there are child packages for handling the specific elements of the interpreter: one for lists, one for elements, one for string-handling, and so forth. The basic layout is as shown: LISP | | \ | \ LISP.Lists | \ | LISP.Elements | \ | LISP.Elements.operations | LISP.Strings
