Function Operable
( Input: List;
Element_Type: Data_Type ) Return Boolean;
Operable tells us that the list can indeed be operated on; every function in LISP is a list, but not every list is a function. Example: What should (1 2 3 4 6) do? On the other hand ("+" 1 2 3 4 5) is simple enough - we apply the function "+" to the tail, which returns 15.
